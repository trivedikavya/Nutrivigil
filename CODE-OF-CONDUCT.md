# Code of Conduct

## Our Pledge

NutriVigil is an open-source project built to improve health awareness using modern technology.  
We are committed to creating a **welcoming, respectful, and inclusive environment** for everyone who participates in this project — regardless of experience level, background, or identity.

We expect all contributors, maintainers, and community members to help foster a safe and constructive space for collaboration.

---

## Our Standards

Examples of behavior that contribute to a positive community include:

- Being respectful, kind, and considerate in language and actions
- Giving and accepting constructive feedback gracefully
- Respecting differing viewpoints and experiences
- Focusing on what is best for the project and its users
- Asking questions and offering help without judgment
- Acknowledging mistakes and learning from them

Examples of unacceptable behavior include:

- Harassment, discrimination, or hateful conduct of any kind
- Personal attacks, insults, or inflammatory comments
- Trolling, intimidation, or deliberately disruptive behavior
- Publishing private information without consent
- Sexualized language or imagery in any context
- Dismissing health-related concerns or misusing AI outputs irresponsibly

---

## Health & AI Responsibility

Because NutriVigil operates in the health and nutrition space:

- Do **not** present AI-generated outputs as medical diagnoses
- Avoid adding misleading, exaggerated, or unsafe health claims
- Respect the disclaimer that NutriVigil is **informational, not medical advice**
- Ensure contributions prioritize user safety, clarity, and ethical AI usage

---

## Scope

This Code of Conduct applies to all project spaces, including but not limited to:

- GitHub issues, pull requests, and discussions
- Code reviews and comments
- Project documentation
- Community interactions related to NutriVigil

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Actions may include:

- Requesting changes or clarification
- Issuing warnings
- Temporarily or permanently removing contributors from the project
- Locking discussions or pull requests when necessary

Enforcement decisions will be made with fairness and the well-being of the community in mind.

---

## Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it responsibly.

You may:
- Open a GitHub issue labeled **Code of Conduct**
- Contact the project maintainer privately (if contact details are provided)

All reports will be handled confidentially and respectfully.

Let’s build it together with empathy, professionalism, and integrity.
