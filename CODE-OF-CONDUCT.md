# Code of Conduct

## Our Pledge

NutriVigil is an open-source project dedicated to improving health awareness through technology.

We pledge to foster a **welcoming, respectful, and inclusive environment** for everyone who contributes to or participates in this project, regardless of experience level, background, or identity.

We are committed to making participation in NutriVigil a harassment-free experience for all.

---

## Our Standards

Examples of behavior that contribute to a positive environment include:

- Using welcoming, inclusive, and respectful language
- Being considerate of differing viewpoints and experiences
- Giving and receiving constructive feedback gracefully
- Collaborating in good faith for the benefit of the project
- Showing empathy towards other community members
- Acknowledging mistakes and learning from them

Examples of unacceptable behavior include:

- Harassment, discrimination, or exclusionary conduct
- Personal attacks, insults, or derogatory comments
- Trolling, intimidation, or deliberately disruptive behavior
- Publishing others’ private information without explicit consent
- Sexualized language or imagery in any project space
- Repeated disregard for community guidelines

---

## Responsible Use in a Health & AI Context

As NutriVigil operates in the health and nutrition domain, contributors are expected to act responsibly:

- Do not represent AI-generated outputs as medical diagnoses
- Avoid misleading or unsafe health-related claims
- Respect that NutriVigil provides **informational content, not medical advice**
- Prioritize user safety, clarity, and ethical considerations in discussions and contributions

These expectations apply to discussions, documentation, and code contributions.

---

## Scope

This Code of Conduct applies within all project spaces, including but not limited to:

- GitHub issues, pull requests, and discussions
- Code reviews and comments
- Documentation and project content
- Any other community interaction related to NutriVigil

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

In response to violations, maintainers may take actions they deem appropriate, including:

- Requesting clarification or corrective changes
- Issuing warnings
- Temporarily or permanently restricting participation
- Locking conversations when necessary

All enforcement decisions will be made with fairness, transparency, and community well-being in mind.

---

## Reporting

If you experience or witness behavior that violates this Code of Conduct, please report it responsibly.

You may:
- Open a GitHub issue labeled **Code of Conduct**
- Contact a project maintainer privately (if contact information is available)

Reports will be reviewed confidentially and respectfully.
